# TJBFight

一个小游戏😁

由pygame编写

## 操作:

* 左键:普通攻击
* 右键:格挡
* 滚轮滚动:精神攻击
* a,w,d:分别控制左,上,下

## 下载:

__在旁边的Releases中下载打包的exe文件__
